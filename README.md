# Deep Learning Projects

This repository is a collection of small deep learning projects and assignments.

## Projects

- [project-01-pendigits-mlp](project-01-pendigits-mlp): Multiclass handwritten digit classification on the UCI Pendigits dataset using NumPy, scikit-learn, and PyTorch multilayer perceptrons.
- [project-02-cifar10-cnn-classification](project-02-cifar10-cnn-classification): CIFAR-10 image classification with custom CNNs, pretrained ResNet18, and a hybrid CNN feature extraction plus Linear SVM pipeline.
